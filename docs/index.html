話すだけで、どこにでも。
ローカル LLM が整える音声入力

VoiceDict は macOS のメニューバーに常駐する音声入力アプリ。話した言葉を端末内の LLM が読みやすい文章に整え、エディタでもチャットでも、フォーカス中のアプリへそのまま挿入します。

ダウンロード GitHub で見る
Developer ID 署名 + Apple ノータライズ済み。ターミナル操作は不要です。
$ 🎤 speak "えーと 明日の 10 時からの会議 リスケでお願いします" → transcribe on-device (SFSpeechRecognizer / WhisperKit) → refine on-device (MLX local LLM) → insert "明日 10 時からの会議はリスケでお願いします。"

実際に動いているところ

VS Code 上で音声入力 → ローカル LLM 整形 → 挿入までの流れ

主な機能

⚡️

ホットキー一発、どのアプリでも

Shift+Option 同時押し・Fn 長押し・カスタムショートカットなど 4 方式。Push-to-talk / トグル / ハンズフリー (VAD) の 3 モード。

🧠

ローカル LLM が文章を整える

フィラー除去・句読点・誤変換の修正を端末内で実行。MLX + 4bit 量子化モデルで Apple Intelligence 不要、完全オフライン動作。

🔀

プロバイダを自由に選べる

STT 4 種 (Apple / WhisperKit / OpenAI / xAI)、整形 LLM 8 種。LiteLLM / Ollama 等の OpenAI 互換エンドポイントにも接続可。

✏️

Edit Mode — 声で書き換え

テキストを選択して「英語に翻訳して」と話すだけで選択範囲を書き換え。生成中のテキストはリアルタイム表示、ESC でいつでも中断。

📚

辞書と履歴

専門用語の置換テーブルを登録、LLM プロンプトにも注入。履歴の保持期間は無効〜無期限から選択。

🔄

自動アップデート

新バージョンはアプリ内 1 クリックで適用 (Sparkle)。更新ファイルは EdDSA 署名で改ざん検証。

プライバシーは、設計で守る

「クラウドに送らない」を選べることが、VoiceDict の出発点です。

✓

完全ローカル動作

ローカル系プロバイダだけを選べば、音声もテキストも端末から一切出ません。

✓

明示的な同意

クラウドサービスの初回利用時は、送信先を明示した同意ダイアログを必ず表示。

✓

Keychain 保存

API キーは端末ローカル限定で Keychain に保存。iCloud には同期しません。

✓

Sandbox 有効

macOS Sandbox 内で動作。要求する権限はマイクとネットワークだけ。
