話すだけで、どこにでも。
ローカル LLM が整える音声入力

VoiceDict は macOS のメニューバーに常駐する音声入力アプリ。話した言葉を端末内の LLM が読みやすい文章に整え、いま使っているアプリへそのまま挿入します。

実際に動いているところ

VS Code 上で音声入力 → ローカル LLM 整形 → 挿入までの流れ

主な機能

ホットキー一発、どのアプリでも

Shift+Option 同時押し・Fn 長押し・カスタムショートカットなど 4 方式。Push-to-talk / トグル / ハンズフリー(発話自動検出)の 3 モード。

ローカル LLM が文章を整える

「えーと」の除去・句読点・誤変換の修正を端末内の LLM (MLX) が実行。Apple Intelligence 不要、API キー不要、完全オフラインで動作。

プロバイダを自由に選べる

文字起こしは 4 種 (Apple / WhisperKit / OpenAI / xAI)、整形 LLM は 8 種。軽量ローカル + 強力クラウドの使い分けも。

Edit Mode — 声で書き換え

テキストを選択して「英語に翻訳して」「もっと丁寧に」と話すだけで、選択範囲を書き換え。生成中のテキストはリアルタイム表示。

辞書と履歴

専門用語の置換テーブルを登録でき、LLM プロンプトにも注入。入力履歴は保持期間を選択可能 (無効〜無期限)。

自動アップデート

新バージョンはアプリ内 1 クリックで適用 (Sparkle)。更新ファイルは EdDSA 署名で改ざん検証されます。

プライバシーは、設計で守る

「クラウドに送らない」を選べることが、VoiceDict の出発点です。

完全ローカル動作

ローカル系プロバイダだけを選べば、音声もテキストも端末から一切出ません。

明示的な同意

クラウドサービスを初めて使うときは、送信先を明示した同意ダイアログを必ず表示。

Keychain 保存

API キーは macOS の Keychain に端末ローカル限定で保存。iCloud には同期しません。

Sandbox 有効

アプリは macOS Sandbox 内で動作。要求する権限はマイクとネットワークだけです。
