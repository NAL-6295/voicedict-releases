## 1. 概要

VoiceDict（以下「本アプリ」）は、ユーザのマイク音声を文字起こしし、テキストフィールドへ挿入する macOS アプリです。本ポリシーは、本アプリがユーザの情報をどのように扱うかを説明します。

**本アプリの基本方針**: ユーザの音声・テキストデータをアプリ提供者（開発者）は一切受信・収集しません。すべての処理はユーザのデバイス内、またはユーザが明示的に選択したクラウドサービス上で完結します。

## 2. 収集しない情報

本アプリは以下の情報を **一切収集しません**:

- ユーザの音声データ
- ユーザが入力 / 挿入したテキスト
- 利用統計、クラッシュレポート、テレメトリ
- アカウント情報、メールアドレス、デバイス識別子
- ネットワーク経由でのアプリ提供者への送信情報

本アプリにはアクセス解析 SDK / 広告 SDK / クラッシュレポータ / アナリティクスのいずれも組み込まれていません。

## 3. ローカルに保存される情報

本アプリは以下をユーザのデバイス内のみに保存します:

| 種別 | 保存先 | 用途 | 削除方法 |
|---|---|---|---|
| 音声認識履歴（生テキスト・整形済テキスト・タイムスタンプ・対象アプリ Bundle ID） | SwiftData（`~/Library/Containers/io.github.nal-6295.VoiceDict/`） | 過去のディクテーション結果の閲覧・再挿入 | Settings → 履歴 → 全削除、または保持期間設定で自動削除 |
| 辞書エントリ（発音 / 表記 / カテゴリ） | SwiftData | LLM 後処理プロンプトに注入 | Settings → 辞書 → 削除 |
| 設定値（プロバイダ選択、ホットキー、システムプロンプト等） | UserDefaults | アプリ動作のカスタマイズ | アプリ削除で全消去 |
| API キー（ユーザがクラウド LLM / STT を使う場合） | Keychain（`kSecAttrAccessibleWhenUnlockedThisDeviceOnly`） | クラウド API 認証 | Settings → プロバイダ → 削除 |

履歴の保持期間（既定: 30 日）は Settings → 履歴 から変更可能です（`disabled / 1 日 / 7 日 / 30 日 / 90 日 / 無期限`）。「履歴を残さない」を選択すれば保存されません。

ディスク全体の保護は macOS の FileVault に依存します。FileVault を有効化することを推奨します。

## 4. 外部送信（ユーザの明示的な選択時のみ）

ユーザが Settings で以下を選択した場合のみ、データが外部に送信されます:

### 4.1 STT プロバイダ

| プロバイダ | 外部送信 | 送信内容 |
|---|---|---|
| Apple SFSpeechRecognizer（既定） | ❌ なし | `requiresOnDeviceRecognition = true` で完全ローカル |
| WhisperKit | ❌ なし | 完全オンデバイス（CoreML） |
| OpenAI Whisper API | ✅ あり | 録音した音声データを OpenAI のサーバへ |
| xAI STT API | ✅ あり | 録音した音声データを xAI のサーバへ |

### 4.2 後処理 LLM プロバイダ

| プロバイダ | 外部送信 | 送信内容 |
|---|---|---|
| 後処理なし（passthrough） | ❌ なし | — |
| NaturalLanguage（既定 ※） | ❌ なし | OS 同梱フレームワーク |
| Apple Foundation Models（既定 ※） | ❌ なし | オンデバイス LLM（Apple Intelligence） |
| MLX ローカル LLM | ❌ なし | 完全オンデバイス（Llama / Qwen / LFM2.5 等） |
| Claude Haiku 4.5 | ✅ あり | 音声認識後のテキスト + ユーザ辞書 + システムプロンプトを Anthropic のサーバへ |
| OpenAI gpt-5-nano | ✅ あり | 同上を OpenAI のサーバへ |
| xAI Grok | ✅ あり | 同上を xAI のサーバへ |
| カスタム（OpenAI 互換） | △ ユーザ指定先 | 同上を **ユーザが Settings で指定したサーバ** へ（ローカルサーバを指定すれば外部送信なし）。取り扱いは指定先サーバの運営者に依存します |

※ 既定はローカル処理のみ: Apple Intelligence が利用可能な環境では Foundation Models、それ以外では NaturalLanguage（Dictation）/ MLX（Edit Mode）が既定になります。

### 4.3 送信先における取り扱い

外部送信されたデータは、各プロバイダのプライバシーポリシーに従って処理されます:

- OpenAI: https://openai.com/policies/privacy-policy
- Anthropic: https://www.anthropic.com/legal/privacy
- xAI: https://x.ai/legal/privacy-policy

ユーザは各社と直接の契約関係（API キー）を持ち、本アプリ提供者はその通信を中継・記録しません。

### 4.4 モデルファイルのダウンロード

WhisperKit / MLX ローカル LLM を選択した場合、初回利用時にモデルファイルを Hugging Face（huggingface.co）からダウンロードします。この通信は **モデルの取得のみ** で、ユーザの音声・テキスト・個人情報は一切送信されません。ダウンロード後の推論は完全にデバイス内で実行されます。

## 5. 必要な権限

本アプリの動作には以下の権限が必要です:

| 権限 | 目的 | 必須 |
|---|---|---|
| マイク | 音声入力 | 必須 |
| 音声認識 | Apple SFSpeechRecognizer 利用時 | 任意 |
| アクセシビリティ | テキストフィールドへの挿入 / 選択範囲取得 | 必須 |
| 入力監視 | ホットキー（Shift+Option / Fn / 任意 1 キー）検出 | カスタムショートカット使用時は不要 |

これらの権限は、ユーザが System Settings で個別に許可・取消できます。本アプリはこれらの権限を使って **音声入力以外の目的でデータを取得することはありません**。

## 6. 子供の利用について

本アプリは 13 歳未満のユーザを対象として設計されていません。13 歳未満のお子様による利用は推奨しません。

## 7. ハンズフリーモードについて

「ハンズフリー（連続 VAD）」モードを有効化すると、ユーザがメニューバーから停止するまでマイクが常時 ON となります:

- メニューバーアイコンが録音中の状態に変化します
- 初回有効化時に確認ダイアログが表示されます
- macOS 標準のマイク使用中インジケータ（橙ドット）も表示されます

ハンズフリーモード中の音声処理範囲は他モードと同一（Settings の STT/LLM 設定に依存）で、本アプリ提供者に送信されることはありません。

## 8. データ削除権

ユーザは以下の方法でデータを削除できます:

- 履歴: Settings → 履歴 → 全削除
- 辞書: Settings → 辞書 → 削除
- API キー: Settings → プロバイダ → 削除
- 全設定 / アプリ全体: アプリを `/Applications` から削除し、`~/Library/Containers/io.github.nal-6295.VoiceDict/` を削除

## 9. ポリシーの変更

本ポリシーは予告なく変更されることがあります。重要な変更がある場合はリポジトリの CHANGELOG または通知ダイアログで告知します。
